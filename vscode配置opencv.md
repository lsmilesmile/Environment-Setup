> Felix
>
> 2022.4.13



用CMake编译好OpenCV源代码后，主要就是vscode中的几个文件要配置OpenCV相关路径：

**tasks.json**

```json
{
    "tasks": [
        {
            "type": "cppbuild",
            "label": "C/C++: g++.exe build active file",
            "command": "E:\\MinGW64\\bin\\g++.exe",
            "args": [
                "-fdiagnostics-color=always",
                "-g",
                "${file}",
                "-o",
                "${fileDirname}\\bin\\${fileBasenameNoExtension}.exe",
                "E:/cpplibrary/opencv/mingwbuild/install/x64/mingw/bin/libopencv_world3410.dll",
                "-I",
                "E:/cpplibrary/opencv/mingwbuild/install/include",
                "-I",
                "E:/cpplibrary/opencv/mingwbuild/install/include/opencv",
                "-I",
                "E:/cpplibrary/opencv/mingwbuild/install/include/opencv2",
                "-L",
                "E:/cpplibrary/opencv/mingwbuild/install/x64/mingw/lib"
            ],
            "options": {
                "cwd": "${fileDirname}"
            },
            "problemMatcher": [
                "$gcc"
            ],
            "group": {
                "kind": "build",
                "isDefault": true
            },
            "detail": "Task generated by Debugger."
        }
    ],
    "version": "2.0.0"
}
```

```json
    "version": "0.2.0",
    "configurations": [
      {
        "name": "g++.exe - Build and debug active file",
        "type": "cppdbg",

        "args": [],

        "cwd": "${fileDirname}",
        "environment": [],
        "externalConsole": false,
        "MIMode": "gdb",

        "setupCommands": [
          {

            "text": "-enable-pretty-printing",
            "ignoreFailures": true

        ],
        "preLaunchTask": "C/C++: g++.exe build active file"

  }
```



**c_cpp_properties.json**

```json
{
    "configurations": [
        {
            "name": "GCC",
            "includePath": [
                "${workspaceFolder}/**",
                "E:\\cpplibrary\\opencv\\mingwbuild\\install\\include",
                "E:\\cpplibrary\\opencv\\mingwbuild\\install\\include\\opencv",
                "E:\\cpplibrary\\opencv\\mingwbuild\\install\\include\\opencv2"
            ],
            "defines": [
                "_DEBUG",
                "UNICODE",
                "_UNICODE"
            ],
            "windowsSdkVersion": "10.0.17763.0",
            "compilerPath": "E:/MinGW64/bin/g++.exe",
            "cStandard": "c17",
            "cppStandard": "c++11",
            "intelliSenseMode": "windows-gcc-x64"
        }
    ],
    "version": 4
}
```



如果代码中包含

```c++
#include <opencv2/opencv.hpp>
```

不报错了，也就是能正常过找到该文件了，说明路径没错。

如果编译好后直接运行生成的.exe文件出现**无法定位程序输入点**，则将libstdc++-6.dll放到和.exe文件相同的位置即可。
